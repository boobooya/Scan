Composite Scoring Formula
// ───────────────────────────────────────────────────────────────────── // EDGE SCORE = Weighted sum of 8 signal components (0–100 each) // Higher score = more undervalued / higher probability edge // ───────────────────────────────────────────────────────────────────── EDGE_SCORE = ( W1 × CONSOLIDATION_SCORE // 15-bar+ contraction → IV not priced for move W2 × KC_BAND_SCORE // Price hugging Keltner band 4–8 bars → coiled W3 × EXPECTED_MOVE_SCORE // Low EM relative to hist volatility → cheap W4 × GAMMA_LEVEL_SCORE // Far gamma walls away → price free to travel W5 × IV_RANK_SCORE // IV Rank < 30 → premiums suppressed W6 × IV_SKEW_SCORE // Flat/inverted skew → directional cheapness W7 × OI_MOMENTUM_SCORE // Rising OI at strike → informed accumulation W8 × TIME_VALUE_SCORE // Theta decay curve position → buy sweet spot ) / SUM(W1..W8) // ── Signal Definitions ──────────────────────────────────────────────── CONSOLIDATION_SCORE = f( ATR_contraction, HV_contraction, bars_since_breakout ) KC_BAND_SCORE = f( price_vs_upper_band, price_vs_lower_band, bars_touching ) EXPECTED_MOVE_SCORE = f( EM/price, EM_vs_HV, growing_EM_wings ) GAMMA_LEVEL_SCORE = f( distance_to_nearest_gamma_wall, net_gamma_exposure ) IV_RANK_SCORE = f( IV_Rank_52w, IV_vs_HV_ratio ) IV_SKEW_SCORE = f( 25d_risk_reversal, skew_direction ) OI_MOMENTUM_SCORE = f( delta_OI_3d, volume_to_OI_ratio, call_put_imbalance ) TIME_VALUE_SCORE = f( DTE, theta_to_premium_ratio, gamma_per_dollar ) // ── Strike Selection Logic ──────────────────────────────────────────── TARGET_STRIKES = { CALL: current_price + (EM_upper × 0.8 .. 1.2) // upper EM boundary ±buffer PUT: current_price - (EM_lower × 0.8 .. 1.2) // lower EM boundary ±buffer FILTER: EDGE_SCORE > 60, delta ∈ [0.15, 0.40], DTE ∈ [21, 60] }
